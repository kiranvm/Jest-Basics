# Jest Crash Course - Unit Testing in JavaScript

## Requirements

1. Node - Version 12 or higher
2. NPM  - Version 6 or higher

## Commands

<code>
	$ npm init -y

	Add jest to the test part of package.json

	$ npm i -D jest

	Testing javascript

	$ npm test

</code>
